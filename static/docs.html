‚ö° Quick Start

1. Get Your API Key

Login to the admin panel and create an API key:

  • Go to Admin Panel
  • Navigate to "API Keys" tab
  • Click "Create New"
  • Enter owner name and expiry days
  • Copy your API key (shown only once!)

2. Make Your First Payment Request

curl -X POST https://your-domain.com/api/v1/qr/generate \
  -H "X-API-Key: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "upi": "merchant@upi",
    "amount": 500,
    "message": "Test Payment"
  }'

3. Display QR Code to Customer

The response includes a base64-encoded QR code image that you can display directly:

<img src="data:image/png;base64,{qr_code}" alt="Payment QR" />

4. Verify Payment Status

curl -X POST https://your-domain.com/api/v1/payment/verify \
  -H "X-API-Key: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "order_id": "BHARAT_ORD_...",
    "merchant_id": "BHARAT_...",
    "merchant_key": "BHARAT_KEY_..."
  }'

üîå API Endpoints

POST /api/v1/qr/generate

Generate a UPI QR code for payment

Request Body:

{
  "upi": "merchant@upi",           // Required: Your UPI ID
  "amount": 500,                   // Required: Amount in ‚Çπ
  "message": "Order #12345",       // Optional: Payment description
  "order_id": "CUSTOM_ORD_123",    // Optional: Custom order ID
  "webhook_url": "https://..."     // Optional: Webhook for status updates
}

Response:

{
  "success": true,
  "order_id": "BHARAT_ORD_1700485678_1234",
  "qr_code": "base64_encoded_image...",
  "amount": 500,
  "status": "PENDING",
  "merchant_id": "BHARAT_xyz...",
  "merchant_key": "BHARAT_KEY_123..."
}
POST /api/v1/payment/verify

Verify the status of a payment

Request Body:

{
  "order_id": "BHARAT_ORD_1700485678_1234",
  "merchant_id": "BHARAT_xyz...",
  "merchant_key": "BHARAT_KEY_123..."
}

Response (if successful):

{
  "success": true,
  "order_id": "BHARAT_ORD_1700485678_1234",
  "status": "SUCCESS",
  "amount": 500,
  "bharatpay_reference": "BHARAT1700485765123456",

}

üíª Code Examples

Python

import requests

API_KEY = "your_api_key_here"
BASE_URL = "https://your-domain.com"

# Generate payment QR
response = requests.post(
    f"{BASE_URL}/api/v1/qr/generate",
    headers={"X-API-Key": API_KEY},
    json={
        "upi": "merchant@upi",
        "amount": 500,
        "message": "Order Payment"
    }
)

payment = response.json()
print(f"Order ID: {payment['order_id']}")
print(f"QR Code: {payment['qr_code'][:50]}...")

JavaScript/Node.js

const apiKey = 'your_api_key_here';
const baseUrl = 'https://your-domain.com';

async function createPayment(upi, amount, message) {
  const response = await fetch(`${baseUrl}/api/v1/qr/generate`, {
    method: 'POST',
    headers: {
      'X-API-Key': apiKey,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ upi, amount, message })
  });
  
  const payment = await response.json();
  return payment;
}

createPayment('merchant@upi', 500, 'Order Payment')
  .then(data => console.log('Payment created:', data.order_id));

PHP

<?php
$apiKey = 'your_api_key_here';
$baseUrl = 'https://your-domain.com';

$data = [
    'upi' => 'merchant@upi',
    'amount' => 500,
    'message' => 'Order Payment'
];

$ch = curl_init("$baseUrl/api/v1/qr/generate");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'X-API-Key: ' . $apiKey,
    'Content-Type: application/json'
]);

$response = curl_exec($ch);
$payment = json_decode($response, true);

echo "Order ID: " . $payment['order_id'];
?>
